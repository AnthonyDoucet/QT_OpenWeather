# Build it yourself
Add a **token.h** file to the root
>#ifndef TOKEN_H<br>
>#define TOKEN_H<br>
><br>
>#define API_TOKEN "*token*"<br>
><br>
>#endif // TOKEN_H<br>

Build with at least QT 5
